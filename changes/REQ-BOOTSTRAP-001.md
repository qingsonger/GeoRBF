# REQ-BOOTSTRAP-001

Added the stage-0 Rust workspace, persistent v1 scope and requirement registry,
mathematical and architecture contracts, ADR baseline, three-platform CI,
repository templates, and the first machine-checkable requirement workflow.

No RBF kernel, linear algebra backend, solver, FFI business API, Python module,
Surfe compatibility layer, or five-model architecture is included.

Independent review tightened radial derivative signs and center limits,
scale-aware CPD rank diagnostics, D=1 normal-mode and angular-cone semantics,
strict-SPD local-mixture prerequisites, and orientation-weight validation. It
also made the registry checker reject malformed schemas, imprecise cycle
diagnostics, and incomplete integrated benchmark obligations, and made the
bootstrap CLI reject extra or non-Unicode arguments without panicking. All
pre-release workspace packages now disable publication, enforced by `xtask`.
